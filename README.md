# 🚀 ShadowFox AIML Internship Projects

This repository contains the projects completed during my internship at **ShadowFox** in the domain of **Artificial Intelligence and Machine Learning (AIML)**.

---

## 📌 Projects Included

### 🔹 1. House Price Prediction
- Built a regression model to predict house prices.
- Performed data cleaning and outlier removal.
- Applied feature selection and exploratory data analysis (EDA).
- Used **Random Forest Regressor** for prediction.
- Achieved **R² Score ≈ 0.55** after tuning.

---

### 🔹 2. Sales Data Analysis
- Performed exploratory data analysis on sales dataset.
- Analyzed monthly trends, product categories, and regions.
- Identified top customers and key business insights.
- Created interactive visualizations using **Plotly**.
